Assignments

  1. Assignment01: Basic HTML

    A basic page that provides some basic information about malware. It doesn't use any CSS code at all to style the page, so think of "baby's first webpage" if you will.

  2. Assignment02: Basic CSS

    A page that showcases a random collection of rubber ducks. It uses CSS code to style up the page so it doesn't look bland and boring. Be sure to read the disclaimer at the very top, very important you read that!

  3. Assignment03: Page Layout

    A page that showcases a rubber duck store, or at least, it's supposed to anyway. It shows different brands of ducks, a section for store favorites based on category, and a small gallery of ducks!

  4. Assignment04: Recreate CSS Page

    A page that attempts to recreate the McCausland College of Arts and Sciences webpage from USC. It may not be 1:1, but I had fun making it!

  5. Assignment05: Javascript Buttons, Functions, and More

    A page that does some javascript stuff. Stuff that happens when a section is clicked and when a button is pressed is included.

  6. Assignment06: Javascript Conditionals

    A page that showcases what can be come with if statements in javascript. It dynamically changes images when a slider is moved and shows the current time.

  7. Assignment07: Loops

    A page that showcases loops in javascript. If you click on the "show scene" button, it should show trees and clouds. If it's between 6pm and 6am, the background will become black and show a moon!

  8. Assignment08: Arrays

    A page that showcases arrays in javascript. It shows a row of dogs with what they looked like before adoption. If you click on their images, a pop-up will appear showing that they look like after adoption.

  9. Assignment09: Classes

    A page that shows what classes do in javascript. The page shows 5 pictures of dog paintings and when you click on them, a popup shows information about the painting like its name, artist, and whether it's framed or not.
